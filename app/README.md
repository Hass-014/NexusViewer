# NexusViewer

## Project Overview
NexusViewer is an Android application developed using Kotlin and Android Studio.

The application authenticates users using an API and loads dashboard data after successful login.

---

## Features

- Login Screen
- API Authentication
- Dashboard Screen
- RecyclerView Data Display
- Details Screen
- Logout Function
- Retrofit Integration

---

## Technologies Used

- Kotlin
- Android Studio
- Retrofit
- RecyclerView
- XML Layouts

---

## API

Login Endpoint:

/sydney/auth

Dashboard Endpoint:

/dashboard/{keypass}

## How to Run

1. Open project in Android Studio
2. Sync Gradle

4. Login
5. View Dashboard
6. Open Details
