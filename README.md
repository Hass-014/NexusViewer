# NexusViewer

## Project Overview
NexusViewer is an Android application developed using Kotlin and Android Studio.

The application allows users to:
- Login to the application
- Retrieve animal information from API
- Display dashboard data
- Open detailed information pages
- Navigate between screens

---

## Features
✔ Login Screen  
✔ API Integration  
✔ Dashboard Screen  
✔ RecyclerView Implementation  
✔ Details Screen  
✔ Logout Function  
✔ Dark UI Design
